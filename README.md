# F1 World Championship Dashboard 

An interactive dashboard built using Python and Looker Studio
analyzing 70+ years of Formula 1 data (1950–2024).

## Live Dashboard
[View Dashboard Here] (https://lookerstudio.google.com/reporting/d28c47b6-23ca-41b2-948b-ba833343d221)

## What's Inside
- **Driver Performance** — Top drivers by wins, podiums, and points
- **Constructor Wars** — Team dominance over the decades
- **Circuits Map** — Every F1 circuit plotted on a world map
- **Qualifying** — Most qualifying appearances by driver

## Tools Used
- Python (pandas) — data cleaning
- Google Sheets — data storage
- Looker Studio — visualization

## Data Source
[Formula 1 Dataset (1950-2024) on Kaggle](https://www.kaggle.com/datasets/melissamonfared/formula-1))
